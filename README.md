# Cutup

## Controls

-   Arrow/swipe up/down: Increase/decrease number of subdivisions.
-   Arrow/swipe right/left: Increase/decrease number of reflections.
-   Space: Pause/play.
-   s: Save the current frame as a PNG.
-   Double-tap: switch camera direction.
